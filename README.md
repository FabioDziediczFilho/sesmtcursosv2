# CSE - Centro de Segurança e Emergências

Este é o portal oficial da **CSE - Centro de Segurança e Emergências**, uma plataforma moderna focada em treinamentos de QHSE (Qualidade, Segurança e Meio Ambiente).

## 🚀 Sobre o Projeto

O portal foi desenvolvido com tecnologia de ponta para oferecer uma experiência fluida, rápida e de alto impacto visual. Focado na venda de treinamentos de NR e consultoria SESMT, o site utiliza um design industrial de alto contraste para transmitir autoridade e confiança.

## 🛠️ Tecnologias Utilizadas

- **React + Vite**: Performance ultra-rápida e carregamento instantâneo.
- **React Router**: Navegação fluida entre páginas (SPA).
- **CSS Vanilla (Custom Design)**: Interface exclusiva desenvolvida do zero.
- **AI Generated Assets**: Imagens cinematográficas criadas sob medida para o setor de segurança industrial.

## 📁 Estrutura de Páginas

- **Home**: Landing page de alto impacto e vitrine de cursos.
- **Treinamentos**: Catálogo expandido de NRs (10, 33, 35, etc).
- **SESMT**: Área dedicada para consultoria técnica e laudos.
- **Contato**: Formulário corporativo e informações de atendimento.
